## Cómo crear tu repositorio (una sola vez)

1. Entra a [github.com](https://github.com) y crea una cuenta con un usuario profesional.
2. Botón **+** → **New repository** → nombre: `bitacora-linux-bash` → **Public** → marca *Add a README file* → **Create repository**.
3. Reemplaza el contenido del `README.md` por esta plantilla y ve creando un archivo por módulo (**Add file → Create new file**).
